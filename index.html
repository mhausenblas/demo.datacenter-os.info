The Datacenter Operating System in action!

Available Demos

Currently, the following demos are available for you to try out:

  • Building a Web application with analytics using Docker and Marathon
  • Event info and countdown
  • Elastic Sentiment Analysis (using Apache Mesos, Marathon and Apache Spark)

How do I get started?

Simply launch a DCOS cluster and try out demos by yourself and if you have any questions, join us on Slack!
